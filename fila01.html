¡Bienvenido! En esta guía, te explicaremos los pasos para instalar Nginx en un servidor Linux y configurarlo con SSL para una comunicación web segura.

Paso 1: Actualizar los Paquetes del Sistema

Antes de instalar Nginx, es importante actualizar la lista de paquetes del sistema y actualizar los paquetes existentes. Utiliza el gestor de paquetes correspondiente a tu distribución de Linux:

    
# Para Ubuntu/Debian
    sudo apt update
    sudo apt upgrade

# Para CentOS/RHEL
    sudo yum update
    sudo yum upgrade

Paso 2: Instalar Nginx

A continuación, deberás instalar Nginx en tu servidor. Nuevamente, utiliza el gestor de paquetes según tu distribución:

    
# Para Ubuntu/Debian
    sudo apt install nginx

# Para CentOS/RHEL
    sudo yum install nginx

Paso 3: Iniciar y Habilitar Nginx

Una vez que Nginx esté instalado, puedes iniciarlo y habilitarlo para que se inicie en el arranque del sistema: 

    
sudo systemctl start nginx
sudo systemctl enable nginx

Paso 4: Configurar Nginx para SSL

Ahora que tienes el certificado SSL, debes configurar Nginx para usarlo. Abre el archivo de configuración de Nginx: 

    sudo nano /etc/nginx/sites-available/default

Encuentra el bloque "server" y realiza los siguientes cambios:

    
server {
        listen 82;
        server_name wedsrv.ifct0509.com;
    }

server {
        listen 443 ssl;
        server_name webesrv.ifct0509.com;

        ssl_certificate /etc/ssl/certs/websrv.crt;
        ssl_certificate_key /etc/ssl/private/websr.key;

      }

Guarda el archivo y sal del editor de texto.

Paso 5: Probar la Configuración de Nginx

Antes de aplicar los cambios, valida la configuración de Nginx para evitar posibles problemas:

    sudo nginx -t

Si la prueba es exitosa, recarga Nginx para aplicar los cambios:

    sudo systemctl reload nginx

Paso 6: Verificar la Configuración de HTTPS

Ahora, visita tu dominio utilizando "https://" en el navegador para verificar que Nginx está sirviendo correctamente tu sitio web a través de HTTPS.

¡Felicidades! Tu servidor Nginx ahora está instalado y configurado con SSL.
